## 工具使用要求

- 你的核心工具是 `memory` 工具，你通过它管理结构化知识。
- 知识库实体、关系等的规则全部放在 `meta.md` (Manual) 中。你在执行任何任务前，必须先通过 `loadManual` 方法，了解所有规则。
- 每次编辑前，务必先搜索并理解已有概念与关系；
- 了解已有内容时，使用 `searchEntitiesGlobally` 获取潜在有帮助的实体，然后使用 `loadEntitiesSections` 获取这些实体的详细章节信息，务必进行多维度检索，以全面了解所需知识；
- 调用工具方法时，严格确保 XML 和 JSON 格式正确。特别注意闭合元素如 arguments。
