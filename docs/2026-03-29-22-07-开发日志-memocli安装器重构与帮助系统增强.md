# 开发日志: memocli 安装器 Python 化与帮助系统增强

## 1. 实现的功能点
- **安装器 Python 化 (`install.py`)**：替换了原有的 `install.sh`。使用 Python `pathlib` 和 `os` 进行更健壮的路径处理和权限设置。
- **帮助系统增强**：
    - **全局帮助**：`memocli --help` 现在会详细解释「参数简化」机制（即在库根目录下自动注入 `--path .`）。
    - **子命令帮助**：`memocli <subcommand> --help` 会在显示底层脚本帮助前，输出一段来自包装器的友好提示，说明可以直接通过 `memocli` 调用。
- **子命令列表过滤**：在显示可用命令时，自动排除了 `install.py` 和 `schema_define.py` 等非工具脚本。

## 2. 代码修改
- **删除**：`memories-off-skill/scripts/install.sh`
- **新增**：`memories-off-skill/scripts/install.py`
- **更新**：`docs/TODO.md` (标记进度)

## 3. 验证结果
- 执行 `memocli --help`：输出正确包含参数简化说明。
- 执行 `memocli stats --help`：正确显示 `[memocli]` 前缀提示及 `stats.py` 的原始帮助。
- 在库根目录下执行 `memocli stats`（无参数）：通过智能注入 `--path .` 成功运行。
