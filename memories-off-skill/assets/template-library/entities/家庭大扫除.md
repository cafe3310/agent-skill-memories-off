# 家庭大扫除

## 整体目标 (Goal)
每周一次的家庭大扫除，确保环境整洁舒适。

## 核心步骤 (Procedure)
1. **清理杂物**: 把沙发和地面的所有零碎物品归位。
2. **除尘处理**: 使用吸尘器处理地毯和死角。
3. **猫砂清理**: 重点清理 [[咩咩]] 和 [[啾啾]] 的猫砂盆，并做一次深度消毒。
4. **拖地擦拭**: 使用温水加入适量消毒液进行全屋拖地。

## 资源消耗 (Resources)
- **工具**: 吸尘器、蒸汽拖把。
- **耗材**: 蓝月亮地板清洁剂、滴露消毒液。
