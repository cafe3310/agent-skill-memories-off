# 品牌信息 (Brand)
渴望 (Orijen) 高蛋白无谷物猫粮，产自加拿大。

# 购买信息 (Purchase)
- **规格**: 5.4kg 包装。
- **参考价格**: 约 500-600 元。
- **购买渠道**: 主要是天猫旗舰店或靠谱的代购。

# 喂食建议 (Feeding)
- **每日喂食量**: 约 60-80g，视体重和运动量增减。
- **注意事项**: 必须保证充足的饮用水，高蛋白猫粮容易让猫口渴。
